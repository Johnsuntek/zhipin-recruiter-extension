# BOSS直聘开发岗位筛选助手

一款基于 AI 的 Chrome 浏览器扩展，帮助 HR/HRBP 在 BOSS直聘 平台上自动评估开发岗位候选人、自动预约电话沟通、自动礼貌告知不合格候选人。

> 所有数据仅保存在本地浏览器中，不会上传到任何服务器。

## 核心功能

### 1. AI 智能评估

- 保存职位 JD，AI 自动提取评估维度（技术栈、经验、学历、项目关键词等）
- 自动扫描候选人列表，提取简历信息
- AI 多维度打分（0-100），给出匹配分析、亮点与风险提示
- 支持批量自动筛选，进度条实时展示

### 2. 自动预约沟通

- 对合格候选人，选择可用日期 + 时间段，AI 自动发送预约消息
- 候选人回复后 AI 智能解析时间意向，自动确认或协商
- 预约状态全程追踪：待处理 → 协商中 → 已确认

### 3. 自动礼貌告知

- 对不合格候选人，一键发送可自定义的礼貌回复模板
- 支持自定义话术，保持专业沟通

### 4. 安全策略

- 模拟真人输入（逐字输入 + 随机延迟）
- 可配置回复延迟范围（默认 3-8 秒）
- 每日消息上限（默认 200 条）
- 超时自动标记（默认 24 小时）

## 界面预览

扩展包含三个主要界面：

| 界面 | 说明 |
|------|------|
| **Popup 弹窗** | 快速查看统计数据、即将到来的预约、已保存职位、最近评估 |
| **侧边面板** | BOSS直聘页面右侧滑出，管理候选人评估、预约、筛选的主操作区 |
| **设置页** | 配置 AI 模型、HR 信息、消息模板、安全策略、数据管理 |

> 运行 `open ui-preview.html` 可在浏览器中查看完整的 UI 渲染预览。

## 安装使用

### 1. 安装扩展

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目文件夹

### 2. 配置设置

1. 点击扩展图标 → 「设置」，或右键扩展图标 → 「选项」
2. 填写 AI 模型配置：
   - **API 服务商**：OpenAI 或自定义（兼容 OpenAI 格式的任何服务）
   - **API Key**：你的 API 密钥
   - **模型名称**：推荐 `gpt-4o` 或同等能力模型
3. 填写 HR 信息（姓名、公司名）
4. 按需调整消息模板和安全策略
5. 点击「保存设置」

### 3. 开始使用

1. 打开 [BOSS直聘](https://www.zhipin.com) 并登录招聘端
2. 进入职位管理页，点击开发岗位旁的「保存 JD」按钮
3. 进入候选人聊天列表，点击右侧「筛选助手」按钮打开侧边面板
4. 选择目标职位，点击「开始自动筛选」
5. 筛选完成后：
   - **合格候选人**：点击「预约沟通」选择时间段
   - **不合格候选人**：点击「拒绝告知」发送模板消息

## 技术栈

- **扩展标准**：Chrome Extension Manifest V3
- **前端**：原生 JS + CSS（无框架依赖）
- **UI 隔离**：Shadow DOM（避免与宿主页面样式冲突）
- **AI 集成**：OpenAI 兼容 API（支持任何兼容格式的 LLM 服务）
- **数据存储**：`chrome.storage.local`（纯本地）

## 注意事项

- 本工具仅面向**开发类岗位**（前端、后端、全栈、DevOps 等）
- 请合理设置安全策略参数，避免触发平台风控
- API Key 仅存储在本地浏览器中，卸载扩展后自动清除
- 建议先小范围测试，确认 AI 评估质量后再批量使用
